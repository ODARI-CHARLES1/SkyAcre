# SkyAcre API Documentation

This directory contains the API documentation for the SkyAcre project.

## Files

- **API-DOC.md**: Detailed API documentation including endpoints, request/response formats, and usage examples.

## Overview

The SkyAcre API provides endpoints for managing users, farmers, and related services in the agricultural platform.

For detailed information, please refer to [API-DOC.md](API-DOC.md).

## Getting Started

1. Review the API documentation in API-DOC.md
2. Set up authentication as described
3. Use the provided endpoints for integration
